# SORCE Collective

A Canadian artist-academic network facilitating connections among creative academics to connect, reflect, and share their practices.

**Live site:** [iowyth.github.io/sorce](https://iowyth.github.io/sorce/)

## About

SORCE Collective generates opportunities for artist-academics to connect, reflect, and share their practices in the hopes of fostering creative collaborations.

## Contributing

This site is collaboratively maintained. There are several ways to contribute:

### Quick Edits
Every page has an "Edit on GitHub" link at the bottom. Click it to propose changes directly.

### Write a Blog Post
1. Fork this repository
2. Create a new file in `_posts/` named `YYYY-MM-DD-your-title.md`
3. Add front matter and content (see existing posts for examples)
4. Submit a pull request

### Add Your Profile
1. Fork this repository
2. Create a new file in `_people/` named `your-name.md`
3. Add your profile info (see `_people/example-member.md`)
4. Submit a pull request

### Propose an Initiative
Open an issue using the "Initiative Proposal" template.

### Suggest a Resource
Open an issue using the "Resource Suggestion" template.

## Local Development

```bash
# Install dependencies
bundle install

# Run local server
bundle exec jekyll serve

# View at http://localhost:4000
```

## Structure

```
_posts/          # Blog posts
_people/         # Member profiles
_initiatives/    # Initiative pages
_layouts/        # Page templates
_includes/       # Reusable components
assets/          # CSS, JS, images
```

## Features

- **Comments** - Giscus-powered discussions on posts
- **Collaborative editing** - Edit any page via GitHub
- **Discussions** - Community conversation via GitHub Discussions
- **Responsive** - Mobile-friendly design
